## 🧠 My Approach to Vibe Coding

Most people think AI-generated code is "cheap."  
I think the **prompt is the craft**.

Here's what I bring to every project:

- **📐 Business Context** — Translating real-world problems into AI-understandable instructions
- **🎯 Domain Expertise** — Injecting field knowledge (data analytics, visualization, UX) into prompts
- **🔁 Iteration** — Refining outputs until they actually *work* in the real world
- **🏗 Architecture Thinking** — Structuring AI-generated code for maintainability and clarity

> Anyone can ask AI to write code.  
> **Not everyone knows what to ask — or why.**
