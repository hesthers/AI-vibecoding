# AI 프롬프트 생성기 (AI Prompt Generator) - 입력 항목 샘플 모음

앱 화면의 실제 사이드바 폼 형식에 맞춰 구성된 3가지 상황별 추천 샘플 데이터입니다. 화면에 보이는 각 칸에 그대로 복사하여 붙여넣으시면 됩니다.

---

## 샘플 1: 데이터 분석가의 보고서 요약 (이미지 기반)
### 1. 역할 및 성격
* **역할 (ROLE)**: 데이터 애널리스트
* **전문성 및 성격 (PERSONA)**: 수치와 팩트에 기반한 객관적인 분석가, 중립적인 시각 유지

### 2. 작업 및 배경
* **작업 유형 (TASK)**: 요약 (Summary)
* **상세 지시사항**:
  - 제공된 장문의 시장 조사 보고서에서 핵심 인사이트 3가지만 추출할 것
  - 요약의 끝부분에는 향후 액션 아이템(Action Item) 1가지를 반드시 포함할 것
* **대상 (TARGET)**: C-레벨 경영진 (CEO, CFO)
* **제약 사항 (CONSTRAINTS)**: 500자 이내로 요약하고, 서술형보다는 명사형으로 끝맺을 것
* **참조 데이터 (REFERENCE)**: (비워두거나 실제 분석할 보고서 텍스트 복사/붙여넣기)

### 3. 출력 형식 & 톤
* **출력 형식**: 표 (Table)
* **톤 (TONE)**: 중립적 (Neutral)
* **응답 언어**: 한국어

### 4. Zero/Few-Shot 예시
* **예시 개수**: 0 (선택하지 않음)

## 샘플 2: 시니어 개발자의 코드 리뷰 요청
### 1. 역할 및 성격
* **역할 (ROLE)**: 시니어 백엔드 개발자
* **전문성 및 성격 (PERSONA)**: 15년 경력, 클린 코드와 성능 최적화의 달인, 매우 논리적이고 직설적인 피드백을 줌

### 2. 작업 및 배경
* **작업 유형 (TASK)**: 코드 리뷰 및 리팩토링 (Code Review)
* **상세 지시사항**:
  - 서론 없이 코드의 문제점만 바로 지적할 것
  - 시간 복잡도(Big-O) 관점에서 비효율적인 부분을 찾아 개선된 코드를 제시할 것
* **대상 (TARGET)**: 1~3년 차 주니어 개발자
* **제약 사항 (CONSTRAINTS)**: 핵심만 3가지로 압축하여 설명할 것, 코드 예시는 반드시 파이썬으로 작성할 것
* **참조 데이터 (REFERENCE)**: (여기에 리뷰를 받고 싶은 작성한 파이썬 코드 블록을 붙여넣기)

### 3. 출력 형식 & 톤
* **출력 형식**: 마크다운 (Markdown)
* **톤 (TONE)**: 전문적이고 논리적인 (Professional)
* **응답 언어**: 영어 (English) (또는 '한국어 + 영어' 선택)

### 4. Zero/Few-Shot 예시
* **예시 개수**: 1
* **요청 예시 (REQUEST) 1**: 이 반복문 로직을 최적화해 줘.
* **원하는 출력 형태 (DESIRED FORMAT) 1**:
  [문제점] 이중 for문으로 인해 O(N^2)의 시간 복잡도가 발생합니다.
  [해결책] Hash Map을 사용하여 O(N)으로 개선할 수 있습니다.
  [개선된 코드] (코드 스니펫 제공)

## 샘플 3: 마케터의 신제품 홍보 카피라이팅
### 1. 역할 및 성격
* **역할 (ROLE)**: 퍼포먼스 마케터
* **전문성 및 성격 (PERSONA)**: 톡톡 튀는 아이디어가 많고, 유머러스하며, 클릭률(CTR)을 높이는 카피 작성의 전문가

### 2. 작업 및 배경
* **작업 유형 (TASK)**: 마케팅 문구 및 카피라이팅 (Copywriting)
* **상세 지시사항**:
  - 인스타그램 광고 소재로 사용할 짧고 강렬한 문구 5가지를 제안할 것
  - 각각의 문구는 '감성형' 2개, '혜택 강조형(할인율 등)' 3개로 구성할 것
* **대상 (TARGET)**: 20~30대 직장인 여성
* **제약 사항 (CONSTRAINTS)**: 이모지를 적극 활용하고, 한 문구당 20자를 넘지 않을 것
* **참조 데이터 (REFERENCE)**: 신제품 무선 텀블러 (보온 12시간 유지, 가격 39,000원, 파스텔 핑크 색상, 기간 한정 20% 할인 중)

### 3. 출력 형식 & 톤
* **출력 형식**: 리스트형 (Bullet Points)
* **톤 (TONE)**: 친근하고 설득력 있는 (Friendly & Persuasive)
* **응답 언어**: 한국어

### 4. Zero/Few-Shot 예시
* **예시 개수**: 2
* **요청 예시 (REQUEST) 1**: 감성형 문구 작성해 줘
* **원하는 출력 형태 (DESIRED FORMAT) 1**: 🌸 당신의 책상 위, 작은 봄을 선물하세요
* **요청 예시 (REQUEST) 2**: 혜택 강조형 문구 작성해 줘
* **원하는 출력 형태 (DESIRED FORMAT) 2**: 🚨 단 일주일! 파스텔 텀블러 20% 할인 중! 🏃‍♀️
